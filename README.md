# Romantic Surprise Webpage

This repository contains a self-contained romantic surprise webpage that is ready for Netlify deployment.

## Files

- `romantic-surprise.html` — the complete single-file interactive page with embedded images and animations.
- `netlify.toml` — optional Netlify configuration for publishing the site from the repository root.
- `.gitignore` — ignores local environment and editor files.

## Deploy to Netlify

1. Create a GitHub repository and push this folder to it.
2. Open Netlify and choose "New site from Git".
3. Connect your GitHub account, select the repository, and set the build settings:
   - Build command: *(leave blank)*
   - Publish directory: `.`
4. Deploy the site.

Once deployed, Netlify will provide a public URL you can share with her.

## Local preview

Just open `romantic-surprise.html` in any browser to preview the page locally.
